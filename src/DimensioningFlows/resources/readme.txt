What this flow does:
--------------------
Includes creates a Provider block which and starts playing of scans from the test scandata file.
The data is filtered, collected and for each trigger period the object dimensions are calculated.

How to demo this flow:
----------------------
Just start it and see the measured results printed on the console.
The recorded data is played resource folder in this project.
The point cloud of the object is visualized in the PointCloud view.
